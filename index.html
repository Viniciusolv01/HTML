Estrutura de um HTML

Voltar

A estrutura básica de um documento HTML consiste em três partes principais: o elemento html, que engloba todo o conteúdo da página; o elemento head, que contém metadados e informações sobre a página (como título, links para arquivos CSS e JavaScript); e o elemento body, que contém o conteúdo visível da página (textos, imagens, etc.).

Doctype HTML:Declara o tipo de documento HTML, informando ao navegador que se trata de um documento HTML5.

Html:É o elemento raiz do documento, envolvendo todo o conteúdo da página.

Head:Contém metadados sobre a página, como: meta charset="UTF-8": Define a codificação de caracteres como UTF-8, que suporta uma ampla variedade de caracteres. meta name="viewport" ...: Configura a visualização da página em dispositivos móveis. title: Define o título da página, que aparece na aba do navegador.

Body:Contém o conteúdo visível da página, como:h1: Define um título de nível 1.p: Define um parágrafo de texto

Tags

Voltar

Tags: São marcadores que indicam o início e o fim de um elemento HTML (ex: p, h1, div). Geralmente, as tags vêm em pares: uma de abertura (ex: p) e uma de fechamento (ex: p).p>

Listas
Voltar
  1. Lista Ordenada1

    2. São usadas para listas onde a ordem dos itens é relevante, como um roteiro ou um ranking. Cada item é precedido por um número, letra ou outro tipo de marcador que indica a posição na lista.

  2. Lista Não Ordenadas2

    3. São usadas para listas onde a ordem dos itens não é importante, como uma lista de compras ou um menu. Cada item é precedido por um marcador, geralmente um ponto.
